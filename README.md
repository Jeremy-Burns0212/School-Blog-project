# School Blog Project

## Overview
This project is a collaborative template website for a school newspaper or blog. It is designed to provide a structured, maintainable platform for publishing and managing stories across multiple categories, with planned role-based accounts, search functionality, and admin controls.

The website is being built with clarity and an organized content workflow in mind, and will serve as a foundation for future development.

## Technology Stack
- **Framework:** ASP.NET Core MVC
- **.NET Version:** .NET 10
- **Frontend:** HTML5, CSS
- **Backend:** C#
- **Authentication:** Identity Framework (for user accounts and roles)
- **Database:** SQL Server with Entity Framework Core
- **Version Control:** Git & GitHub

## Article Page Categories
- **Graduate Stories** — Articles about alumni
- **Student Stories** — Content focused on current students
- **Faculty & Staff** — Features, interviews, and announcements
- **News** — General news relevant to the school/community

## Homepage Features
- Short list of latest stories
- Short list of featured stories
- Navigation links to all page categories
- Search functionality to find articles by keyword

## Account System
The site includes four account types:
- **Admin** — Full access to all content, user management, and site settings (school info, color scheme, etc.)
- **Editor** — Review and approve stories submitted by writers
- **Writer** — Create and submit stories for review
- **Reader** — Browse and read published content

## Current Status
Project is in active development. The basic structure of the website is being established, with initial focus on setting up the MVC architecture, database models, and user authentication system. Future milestones include implementing the article submission workflow, role-based access control, and search functionality.
